# Pêndulo Simples
O pêndulo simples é um sistema mecânico composto por uma massa pontual presa a uma haste sem massa, movendo-se sob a influência da gravidade.
</br>Este é um algoritmo simples que esta sendo implementado em C# para calcular propriedades de um pêndulo simples. </br>
## Objetivo

O objetivo deste programa é calcular diversas propriedades do pêndulo simples, como o período, a frequência, a velocidade angular, a amplitude máxima e a energia potencial gravitacional.

## Fórmulas Utilizadas

1. **Período (T):** \( T = 2\pi \sqrt{\frac{L}{g}} \)
2. **Frequência (f):** \( f = \frac{1}{T} \)
3. **Velocidade Angular (\(\omega\)):** \( \omega = \frac{2\pi}{T} \)
